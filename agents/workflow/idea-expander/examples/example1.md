# Example: Idea Expansion Output

## Input

음성 인식 기반 메모 앱

## Expansion Report

### Original Idea Summary

음성으로 메모를 기록하고 텍스트로 변환하여 저장하는 모바일/웹 앱

### Expansion Results

#### Direction 1: AI 비서 통합형

- **Description**: 단순 메모를 넘어 음성 명령으로 일정 추가, 리마인더 설정, 할일 생성까지 처리
- **Lens Used**: Combination (기존 캘린더/투두앱과 결합)
- **Potential**: High
- **Key Insight**: 메모 앱을 "음성 기반 생산성 허브"로 확장
- **Development Effort**: High

#### Direction 2: 회의록 특화

- **Description**: 회의 녹음 → 자동 요약 → 액션 아이템 추출 → 참석자별 공유
- **Lens Used**: Scale (개인용 → 팀용 확장)
- **Potential**: High
- **Key Insight**: B2B 시장 진입 가능, 높은 지불 의향
- **Development Effort**: Medium

#### Direction 3: 언어 학습 보조

- **Description**: 음성 메모의 발음 분석, 문법 교정, 다국어 번역 기능 추가
- **Lens Used**: What-If (학습자가 사용한다면?)
- **Potential**: Medium
- **Key Insight**: 언어 학습 시장은 크지만 경쟁 치열
- **Development Effort**: High

#### Direction 4: 시각 장애인 특화

- **Description**: 접근성 최적화, 햅틱 피드백, 음성 네비게이션, 스크린리더 완벽 지원
- **Lens Used**: Inversion (시각 대신 청각/촉각 중심)
- **Potential**: Medium
- **Key Insight**: 사회적 가치 + 정부/기업 CSR 예산 타겟
- **Development Effort**: Medium

#### Direction 5: 감정/건강 추적

- **Description**: 음성 톤 분석으로 감정 상태 추적, 멘탈 헬스 저널링
- **Lens Used**: Analogy (헬스케어 앱의 추적 패턴 적용)
- **Potential**: Medium
- **Key Insight**: 웰니스 트렌드와 맞물림, 프라이버시 이슈 주의
- **Development Effort**: High

#### Direction 6: 오프라인 퍼스트

- **Description**: 완전 로컬 처리, 인터넷 없이 동작, 프라이버시 극대화
- **Lens Used**: Inversion (클라우드 의존 반대)
- **Potential**: Low-Medium
- **Key Insight**: 프라이버시 민감 사용자 니치 존재
- **Development Effort**: Medium

### Cross-Cutting Themes

1. **AI 활용 심화**: Direction 1, 2, 5에서 공통 - 단순 STT를 넘어 지능형 처리
2. **특정 사용자 그룹 타겟팅**: Direction 3, 4에서 공통 - 범용보다 특화가 차별화
3. **프라이버시 고려**: Direction 5, 6에서 공통 - 음성 데이터 민감성

### Top 3 Recommendations

1. **Most Promising**: Direction 2 (회의록 특화) - B2B 시장, 명확한 지불 의향, 기술적 실현 가능
2. **Most Innovative**: Direction 5 (감정/건강 추적) - 새로운 카테고리 창출 가능성
3. **Quickest Win**: Direction 1 (AI 비서 통합) - 기존 기능 확장, MVP 빠른 구현

### Areas for Critical Review

- **Direction 2**: 경쟁 제품(Otter.ai, Fireflies) 대비 차별화 필요
- **Direction 5**: 의료기기 규제, 프라이버시 법규 검토 필요

### Handoff to Critic Agent

Key questions to address:
1. 회의록 시장의 포화도와 진입 장벽
2. 음성 감정 분석의 기술적 정확도 현황
3. B2C vs B2B 전략 선택의 영향
