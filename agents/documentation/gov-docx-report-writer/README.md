# Government Docx Report Writer

## Overview

출장보고서·결과보고·추진계획·검토보고 등 **임의의 문서를 정부 공문서(제안서) 양식의 Word(.docx)로 생성**하는 한국어 전문 에이전트입니다.

이 에이전트의 핵심은 **스타일과 내용의 분리**입니다. 정부 공문서 양식(글꼴·색상·표제부·절 제목 띠·글머리 위계·표·쪽번호·끝.)은 재현이 결정적이므로 번들 스크립트 `scripts/build_gov_docx.py`가 100% 책임지고, 에이전트(LLM)는 내용을 **개조식·명사형**으로 구조화하여 JSON 스펙으로 주입합니다. 따라서 매 문서가 동일한 정부 양식으로 일관되게 출력됩니다.

## Category

`documentation` - 정부 공문서 양식 문서 자동 생성

## Prerequisites

- Python 3 설치
- `python-docx` 패키지 (`python -m pip install python-docx`) — 미설치 시 에이전트가 자동 설치 시도
- Microsoft Word 또는 호환 뷰어(한글/LibreOffice 등)에서 .docx 열람

## 적용 스타일 (정부 공문서 스타일 가이드)

| 항목 | 적용 내용 |
|---|---|
| **글꼴** | 제목·글머리기호·표 머리글 = `맑은 고딕`, 본문 = `바탕`(명조) |
| **색상** | 글자 검정 단색(강조색 없음), 표 머리글 `#E7E6E6`, 라벨 열 `#F2F2F2`, 절 제목 띠 `#D9D9D9`, 테두리 `#808080` |
| **표제부** | ①기관·부서명(굵게)+굵은 가로줄 → ②가운데 큰 제목 → ③'- 부제 -' → ④이중 가로줄 → ⑤우측 작성일·작성자 |
| **절 제목** | 'N. 제목' 형식, 회색 음영 띠 + 왼쪽 굵은 검정 세로바 + 굵게 |
| **글머리 위계** | □(대항목) → ○(중항목) → -(소항목) → ·(세부) |
| **문체** | 개조식·명사형 종결("~함","~필요","~예정","~검토","~확인") |
| **표** | 얇은 회색 테두리, 머리글 행 회색·굵게·가운데, 라벨 열 연회색·굵게 |
| **기본 구성** | 1.추진 개요(표) → 2.추진 배경 및 필요성 → 3.추진 내용 → 4.주요 성과 → 5.시사점 및 향후 계획(표) → 붙임 |
| **마무리** | '붙임 N. ○○○' 표기 후 본문 끝에 '끝.' |
| **쪽번호** | 페이지 하단 가운데 '- N -' |
| **사실성** | 불확실 수치·외부 사실은 출처 괄호 병기, 추정은 '(추정)' 명시, 자료 없으면 창작 금지 |

## Capabilities

- 정부 공문서(제안서) 스타일 .docx 결정적 렌더링
- 표제부 5요소 자동 구성
- 절 제목 회색 띠 + 세로바
- 글머리 위계(□○-·) 자동 적용
- 표 스타일링(머리글 회색·라벨 열 연회색·회색 테두리)
- 표준 5절 구성 + 붙임 + 끝.
- 개조식·명사형 문체 변환
- 하단 가운데 쪽번호
- 출처 병기·추정 명시·사실 창작 금지
- 부분 수정 재렌더링

## Usage

### Basic Usage

```
6/5 ○○산단 △△전자 스마트공장 점검 출장보고서를 정부 공문서 양식 docx로 만들어줘
```

에이전트가 부족 항목(참석자·기관/부서명·성과 출처 등)을 확인한 뒤, 5절 구조로 구조화하여 .docx를 생성합니다.

### 동작 흐름

1. 문서 종류 + 담을 내용 수집 (부족 시 질문, 자료 파일은 정독)
2. 내용을 표준 5절 구조 + 개조식·명사형으로 구조화 (출처 병기·추정 명시·창작 금지)
3. JSON 스펙(`spec.json`) 작성
4. `python-docx` 확인/설치 후 렌더링
5. 결과 파일 경로 + 추정/출처 요약 안내, 수정 요청 대기

### 스크립트 직접 실행 (수동)

```bash
python scripts/build_gov_docx.py examples/sample_spec.json output.docx
```

## JSON 스펙 구조

`examples/sample_spec.json` 참조. 핵심 요소 타입:

| type | 용도 | 주요 필드 |
|------|------|----------|
| `section` | 절 제목(회색 띠+세로바) | `number`, `title` |
| `bullet` | 글머리 항목 | `level`(1~4 → □○-·), `text` |
| `para` | 일반 문단 | `text`, `indent_cm` |
| `table` | 표 | `header_row`(opt), `rows`, `label_col`(bool), `col_widths`(cm) |
| `attachment` | 붙임 | `text` |
| `end` | '끝.' | - |
| `spacer` | 빈 줄 | - |

## Notes

- 스타일을 바꾸려면(예: 색상 코드, 여백) `scripts/build_gov_docx.py` 상단의 스타일 상수를 수정합니다.
- 글머리 위계는 `level` 값(1~4)으로 제어되며 마커(□○-·)는 스크립트가 부여합니다.
- 출처가 불명확한 수치는 반드시 괄호로 출처를 병기하고, 추정은 `(추정)`으로 표기합니다. 자료가 없으면 빈 항목으로 두고 "확인 필요"를 표기합니다.
